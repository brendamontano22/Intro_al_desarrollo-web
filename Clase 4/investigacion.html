Diferencia de etiquetas

La diferencia entre b y strong es que a pesar de que ambas ponen la letra en negritas, si se usa strong se le da un significado semántico, es decir, se considera que el texto es relevante. Cuando se usa b, únicamente cambia el estilo visual.

Ejemplo

Este texto únicamente está en negritas.
Este texto está en negritas pero también es relevante.

En el caso de i y em, es relativamente lo mismo pero en cursiva. Es decir, la i únicamente cambia el estilo visual y la em, además de cambairlo indica énfasis semántico.

Ejemplo

Usando la i cambiamos a cursiva.
En este caso agregammos importancia.

La diferncia entre s y del es nuevamente de forma y significado. s muestra el texto con una línea tachada, que únicamente tiene un efecto de estilo. Cuando se usa del, además de mostrar el texto tachado, indica que fue eliminado de un documento, ayuda a entender cambios en el mismo.

Ejemplo

Precio anterior: $150 Nuevo precio: $180
El presupuesto sería de $10,000 $12,500.

Elementos de línea y bloque

En html las etiquetas se clasifican en elementos de bloque y elementos de línea.

Elementos de bloque

Los elementos de bloque ocupan todo el espacio disponible, es como si fuera una caja que se apila debjao de otra. Cada elemento empieza en un nuevo renglón.
Algunos ejemplos son: div, p, section, article, header, footer.

Elementos de línea

Los elementos de línea ocupan solo el espacio necesario para su contenido y no generan un salto de línea, se colocan uno a lado del otro.
Ejemplos: a, strong, em, img, b, i.

Campos en formularios

  1. Input: recibe información del usuario. Atributos principales: type, name, id, value, placeholder, required, min, max, step.
  2. Label: describe un campo del formulario y mejora la accesibilidad. Atributo principal: for.
  3. Output: representa el resultado generado por el formulario. Atributos: name, for.
  4. Progress: muestra el progreso de una tarea. Atributos: value, max.
  5. Select: crea un menú desplegable. Atributos: name, id, required.
  6. Option: define cada opción dentro del menú. Atributos: value, selected.
  7. Textarea: campo de texto multilínea. Atributos: rows, cols, placeholder.

Un caso especial del campo input es la propiedad type. Sus 7 usos distintos son: text, password, email, number, date, radio, checkbox.
A continuaicón se muestra un formulario en el que se usan los campos descritos anteriormente.
